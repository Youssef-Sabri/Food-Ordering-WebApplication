# FoodDash - Food Ordering Web Application

A full-stack food ordering prototype built with Next.js, SQLite, and Prisma.

## Quick Start

```bash
npm install
npx prisma migrate dev
npx prisma db seed
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
